Key Features

HDR Passthrough

Native HDR passthrough for compatible displays with automatic HDR-to-SDR tonemapping for SDR screens. No washed-out colors.

Skip Intro Detection

Automatic intro and outro detection with on-screen skip notifications. Works like Netflix for most shows.

Smart Track Selection

Intelligent audio and subtitle selection based on preferences. Filters unwanted tracks and handles forced subtitles automatically.

Enhanced Metadata

Multiple rating sources, full cast & crew, network badges, localized content in your preferred language, and episode descriptions.

UI-Driven Settings

All player settings, track preferences, and features configurable via UI. No config file editing required.

Anime Enhancements

Optional Anime4K upscaling for razor-sharp visuals and SVP interpolation for smooth 48/60 FPS motion, automatically applied to anime content.

Frequently Asked Questions

Is this compatible with my existing Stremio account?

Yes. Stremio Kai is a fork of Stremio Community Edition and uses the same account system. Your addons and settings sync automatically.

Why Windows only?

The build relies on WebView2 and Windows-specific MPV optimizations. Cross-platform support would require significant architectural changes.

Will my anime enhancements apply to regular movies?

No. All anime-specific features (Anime4K, SVP) are intelligently detected and only apply when anime content is identified.

Do I need to edit config files?

No. Starting with v4.0, all settings are accessible via the UI. Advanced users can still edit configs manually if desired.
